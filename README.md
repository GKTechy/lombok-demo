# lombok-demo
Lombok using in spring Boot Project

Lombok Download Link:
https://projectlombok.org/download

Install lombok in eclipse using cmd prompt:
java -jar lombok.jar

Lombok Annotation:

  @Data
	@Getter/@Setter
	@ToString
	@EqualsAndHashCode
	@NoArgsConstructor, @RequiredArgsConstructor and @AllArgsConstructor
